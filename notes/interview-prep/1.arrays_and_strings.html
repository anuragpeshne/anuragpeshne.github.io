1 Hash Tables

1.1 Using arrays

  • Create hash, which is int or long using a hashing function.
    • since number of ints is less than the keys, multiple keys will map to same integer
  • Calculate index into array using the hash modulo size of array.
    • Multiple hash will obviously clash to the same index
  • Handle collision using a linked list
  • On average case: O(1)
    • Worst Case: O(N)

3 Questions

3.1 IsUnique

determine if string has all unique characters. What if cannot use additional data structures.

3.1.1 Ans1: Hash Table

We memorize the characters seens so far using a hash table. For each character in string, we mark the spot. If we encounter same character again, we return false.

  1. Python
    def all_unique_char(inp):
   memory = {}
   for c in inp:
       if c in memory:
           return False
       else:
           memory[c] = True

   # we are out of the loop, thus no repeated character found
   return True

inp = "test input string"
print(all_unique_char(inp))
  2. C++
    #include <iostream>
#include <unordered_map>

int main () {
    std::string inp ("test input string");

    std::cout << all_unique_char (inp) << std::endl;
    return 0;
}

bool all_unique_char (std::string& inp) {
    std::unordered_map<char, bool> memory;
    for (auto & c : inp) {
        if (memory.find(c) == memory.end()) {
            memory[c] = true;
        } else {
            return false;
        }
    }
    return true;
}

    Notes:

    • Here c has type char. So for (auto & c : inp) is same as for (char & c : inp).
    • The loop can be written in a traditional way: for (int i = 0; i < memory.length(); i++), then access element memory[i].

3.1.2 Ans2: ASCII array

Instead of using a hash function to map characters, we can use a integer array which index ASCII code to frequency of characters. We can index just alphabets and numbers or simply use a 256 size array to index all characters in 1 byte character.

  1. Python
    def all_unique_char(inp):
    memory = [0] * 256
    for c in inp:
        if (memory[ord(c)] > 0):
            return False
        else:
            memory[ord(c)] = 1

    return True

print(all_unique_char("this is test"))
